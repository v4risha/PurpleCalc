How to use

  • Type directly into the display or use the calculator buttons.
  • Use ^ for powers, for example 5^3.
  • For button-based advanced operations, type comma-separated inputs like 12,18 for gcd/lcm or 2,25 for root.
  • Typed functions also work, like sqrt(25), sin(30), nCr(5,2), and avg(2,4,6).
  • Press Enter for equals, Backspace for delete, and Delete or Escape for AC.
